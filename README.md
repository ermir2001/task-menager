\## Tech stack

\- Frontend: Vue 3 + Vite

\- Backend: FastAPI

\- Database: PostgreSQL

\- Deployment: Vercel / Render



\## Run locally

\### frontend

cd frontend

npm install

npm run dev



\### backend

cd backend

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt

uvicorn app:app --reload
